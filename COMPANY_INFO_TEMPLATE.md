# 🏢 美国公司信息模板

**请按照以下格式填写您的美国公司信息：**

---

## 📝 需要填写的信息

```
公司法定名称（Legal Name）：
示例：Gempix2 LLC 或 Gempix2 Inc.
您的公司名称：_______________________


EIN（雇主识别号）：
格式：XX-XXXXXXX（9位数字，如：12-3456789）
您的 EIN：_______________________


注册州（State of Incorporation）：
示例：Delaware, California, Nevada
您的注册州：_______________________


注册地址（Registered Address）：
完整地址，包括街道、城市、州、邮编
您的注册地址：_______________________


办公地址（Office Address，如果与注册地址相同可不填）：
您的办公地址：_______________________


联系电话（可选）：
您的电话：_______________________


公司成立年份（可选）：
您的成立年份：_______________________
```

---

## 📌 填写示例

```
公司法定名称：Gempix2 LLC
EIN：12-3456789
注册州：Delaware
注册地址：1234 Innovation Way, Suite 100, Wilmington, DE 19801
办公地址：（同上）
联系电话：+1 (555) 123-4567
公司成立年份：2024
```

---

## ⚠️ 重要提示

1. **EIN 格式**：必须是 XX-XXXXXXX 格式（两位数-七位数）
2. **地址**：必须是完整的美国地址
3. **真实信息**：所有信息必须真实，Google 会核实
4. **隐私保护**：
   - EIN 会公开显示（这是美国公司的公开信息）
   - 如果担心隐私，可以使用注册代理地址（Registered Agent Address）
   - 电话可以选填
